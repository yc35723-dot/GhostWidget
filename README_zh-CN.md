**GhostWidget** 是一款专为 Android 设计的“隐形”桌面小组件。旨在满足那些希望在桌面或锁屏上放置载具条码/电子票券/Instagram 个人主页二维码等内容，但又不想遮挡壁纸的用户需求——仅在需要时现身！默认情况下它完全透明，只有在你单击或双击时才会显示你的图片。

这款工具尤其适合追求桌面简洁，或需要在锁屏界面（配合 Samsung Goodlock LockStar）快速出示条码的用户。

---

## ✨ 特色功能

- **完全透明**：平时完全隐藏在桌面背景中，不破坏你的精心布局。
- **单击/双击双图设定**：支持为“单击”和“双击”分别设置不同的图片，灵活度高。
- **可调节响应速度**：经过优化的点击判定逻辑，支持自定义 **150ms - 1000ms** 的反应延迟。
- **屏幕常亮支持**：开启后，只要图片正在显示，屏幕就不会自动熄灭，方便扫描条码。（在 Samsung Goodlock LockStar 中放置于锁屏时不支持）
- **Material You 动态取色**：界面主题会自动跟随系统壁纸颜色变化（Android 12+）。
- **高质量显示**：不缩放、不压缩你的原始图片，确保条码细节清晰可见。

## 🚀 使用指南

1. **设置图片**：打开应用，分别为“单击图片”和“双击图片”选择你想要显示的内容。
2. **添加小组件**：回到桌面，长按空白处 -> 选择“小组件” -> 找到 **GhostWidget** 并拖拽至桌面。
3. **调整反应速度**：在应用内使用滑块调节最适合你的点击节奏（建议设为 200ms，150ms 最快）。
4. **操作小组件**：
   - **单击**：显示第一张图片 / 再次单击隐藏。
   - **双击**：显示第二张图片 / 再次双击隐藏。

## 📱 进阶应用：Samsung Good Lock (LockStar)

对于三星用户，你可以通过 **Good Lock** 中的 **LockStar** 插件，将 GhostWidget 添加到“锁屏”界面。这样你无需解锁手机，只需在锁屏界面点击，即可立即显示载具条码，是强迫症与效率追求者的完美方案！

## 🛠 技术规格

- **开发语言**：Kotlin
- **UI 框架**：Android XML (Material Design 3)
- **最低支持版本**：Android 7.0 (API 24)
- **核心技术**：
  - `AppWidgetProvider` 处理小组件状态。
  - `Static Handler` 确保精确的双击拦截。
  - `WakeLock Service` 实现显示期间屏幕常亮。
  - `Material Design` & `Dynamic Colors` 界面。
